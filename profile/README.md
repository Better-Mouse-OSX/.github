## Overview

<a href="#">BetterMouse</a> addresses the frustration Mac users experience with third-party mice. macOS mouse handling is optimized for Apple's Magic Mouse and trackpad, leaving Logitech, Razer, and other mice feeling sluggish or imprecise. BetterMouse intercepts mouse input and applies customizable acceleration, enabling any mouse to feel natural on Mac.

The <a href="#">acceleration curve editor</a> provides visual control over pointer response. See the relationship between mouse movement speed and cursor travel. Adjust the curve to match your preferences - linear for precision work, accelerated for large displays. This visual approach makes finding your perfect setting intuitive rather than guessing at numbers.

<a href="#">Scroll wheel customization</a> transforms choppy scrolling into smooth motion. Mac's default scroll handling for non-Apple mice often feels stepped and jerky. BetterMouse smooths scroll input for fluid page movement. Adjust scroll speed independent of system settings. Reverse scroll direction if you prefer non-natural scrolling for mice while keeping natural for trackpad.

The <a href="#">button mapping</a> assigns functions to extra mouse buttons. Middle click can open links in new tabs, trigger Mission Control, or run custom actions. Side buttons might control volume, switch spaces, or activate application-specific shortcuts. These mappings unlock utility from buttons macOS ignores by default.

<a href="#">Per-application profiles</a> adjust settings automatically based on active software. Slower, precise movement for photo editing. Faster acceleration for general browsing. Different button mappings for specific applications. BetterMouse switches profiles seamlessly as you move between apps.

The <a href="#">pointer speed</a> adjustment separates from macOS system preferences. Set speeds beyond Apple's maximum limit. Fine-tune at granular levels system settings don't offer. Maintain consistent cursor behavior regardless of macOS preference changes.

<a href="#">Polling rate optimization</a> ensures responsive tracking for gaming mice. Some mice report position updates faster than macOS expects. BetterMouse handles high-frequency input correctly, translating superior mouse hardware into improved cursor responsiveness.
